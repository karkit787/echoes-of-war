# Chapter 0 Scene Setup Extension

Project extension for Cocos Creator 3.8.8.

With `assets/scenes/Chapter0.scene` open, use:

- **Echoes of War > Chapter 0 > Build and Bind Chapter0.scene**
- **Echoes of War > Chapter 0 > Validate Chapter0.scene**

The build action mutates and saves the scene through Cocos' scene process. It
does not write serialized scene JSON directly. See
`docs/dev/chapter0_cocos_scene_setup.md` for the generated hierarchy,
validation coverage, and recovery steps.
